# 智能档案管理系统（后端工程骨架）

本目录已生成后端 Spring Boot（Java 17）最小可运行骨架：启动类与 `application.yml`。

## 你需要补齐的内容
- 在 `pom.xml` 中把 `spring.boot.version`、`mybatis.plus.version` 从 `CHANGE_ME` 修改为你们项目标准版本
- 配置 PostgreSQL 连接信息（`application.yml` 中 `spring.datasource`）
- 安装 Java 17 + Maven 后执行 `mvn package` / `mvn spring-boot:run`
